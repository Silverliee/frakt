Fractal Library

A library for generating fractal images using various algorithms.

Algorithms

The library provides implementations for the following fractal algorithms:

  • Julia Set
  • Mandelbrot Set
  • Iterated Sin(z)
  • Newton-Raphson (z^3)
  • Newton-Raphson (z^4)
  • Nova Newton-Raphson (z^3)
  • Nova Newton-Raphson (z^4)

Color Generation

The library also provides a color function for mapping a continuous parameter t to an RGB color.

Image Creation

You can create fractal images using the create_image function, which takes a FragmentTask and a vector of PixelIntensity. The images are saved to the specified path, and the file name is generated randomly.
